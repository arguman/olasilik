# Description
This module functions returns a lot of lucky games.

## Options

- Dice Roll
- Head & Tail
- Rock & Paper & Scissors

## Simple Usage

Instantiate `Possibility` class, rock & roll.

    o = Possibility()
    o.dice_roll(2)
    o.head_tail()
    o.rock_paper_scissors()
